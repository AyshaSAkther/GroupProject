This project is a web-based donation platform built using PHP, MySQL, HTML, and CSS. It allows users to register, log in, create fundraising campaigns, and donate to different causes such as education, medical support, children, and immigration.

The goal of this project is to demonstrate full-stack development, including database design, backend processing, and the use of SQL joins to connect and display related data across multiple tables.

What You Will See in the Report

The project report explains the full development process, including:

How the EER diagram was designed
How the database was created in phpMyAdmin
How SQL code was written manually
How SQL joins were implemented and used
Explanation of each file and its role in the system
Challenges faced during development and how they were solved
Security considerations and improvements
